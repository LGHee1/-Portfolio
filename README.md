# 🏃‍♂️ RunningApp  
위치 기반 러닝 · 산책 경로 기록 및 추천 모바일 애플리케이션

---

## 📌 프로젝트 개요
RunningApp은 사용자의 위치 정보를 기반으로  
러닝 및 산책 활동을 기록하고,  
다른 사용자들이 실제로 달린 경로를 공유·추천받을 수 있는 모바일 앱입니다.

익숙하지 않은 장소에서도 안전하고 효율적인 러닝을 할 수 있도록  
**실시간 트래킹 + 경로 공유 + 추천 시스템**을 목표로 개발했습니다.

---

## 🎯 기획 의도
- 혼자 러닝할 때 **어디를 달려야 할지 모르는 문제**
- 여행지나 새로운 동네에서 **검증된 러닝 코스 부족**
- 단순 기록 앱이 아닌 **사용자 참여형 러닝 플랫폼** 필요

➡️ 실제 사용자가 달린 코스를 기반으로  
➡️ 추천 수에 따라 신뢰도 있는 경로 제공

---

## 🛠 주요 기능

### 1️⃣ 실시간 러닝 트래킹
- GPS 기반 실시간 위치 추적
- 운동 시간, 이동 거리, 평균 페이스 계산
- 칼로리 소모량 자동 계산
- 실시간 데이터 업데이트

### 2️⃣ 운동 기록 저장
- 러닝 종료 시 운동 데이터 저장
- 날짜별 운동 기록 관리
- 운동 요약 화면 제공

### 3️⃣ 경로 공유 기능
- 사용자가 직접 달린 러닝/산책 경로 공유
- 경로에 사진 및 설명 추가 가능

### 4️⃣ 추천 경로 시스템
- 다른 사용자의 추천(좋아요) 수 기반 정렬
- 현재 위치 기준 근처 러닝 코스 추천

### 5️⃣ 사용자 인증
- 회원가입 / 로그인 기능
- 개인 운동 기록 분리 관리

- **개발 환경**: Android Studio
